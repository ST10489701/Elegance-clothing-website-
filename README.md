- Semantic HTML5 structure

- External CSS stylesheet
- CSS Grid and Flexbox layouts
- Typography styling with font properties
- Responsive design (mobile, tablet, desktop)
- Media queries and breakpoints
- Interactive hover effects
- Visual styling with colors and shadows

#### Part 3 - JavaScript & SEO ✅
- Interactive search functionality
- Image lightbox/modal gallery
- Two HTML forms (Enquiry & Contact)
- JavaScript form validation
- Error handling and success messages
- SEO optimization (meta tags, keywords, alt text)
- Smooth scrolling navigation
- Dynamic content loading
